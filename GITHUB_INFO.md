## 🎯 Co program dělá

Program zpracovává data z alpských lyžařských závodů:

```
VSTUP (soubor)
    ↓
Načtení + Parsování
    ↓
Transformace (velká písmena)
    ↓
Vyhledávání
    ↓
Řazení (quicksort)
    ↓
Zápis do souboru
    ↓
VÝSTUP (terminál + soubor)
```
